/api/vehiclesInLot
GET: all vehicles in lot
POST: add new vehicle to lot

GET: find vehicle by id
PUT: update vehicle by id
DELETE: delete vehicle by id

/api/vehiclesInLot/:plate
GET: find vehicle by plate
PUT: update vehicle by plate
DELETE: delete vehicle by plate
